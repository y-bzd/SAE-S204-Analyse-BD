# 📊 Analyse de Données - Industrie du Jeu Vidéo

**Auteur :** Youcef BOUZID  
**Technologies :** Python (Jupyter Notebook), SQL, Pandas, Matplotlib  
**Contexte :** SAE S2.04 - Exploitation d'une base de données (BUT Informatique)

## 📝 Présentation du Projet
Ce projet consiste en l'exploration et l'analyse d'une base de données relationnelle portant sur l'industrie mondiale du jeu vidéo. L'objectif est d'extraire des statistiques pertinentes sur les plateformes (consoles), les éditeurs et les studios de développement via des requêtes SQL complexes, puis de visualiser ces données avec Python.

## 📂 Structure du dépôt
* `analyse_donnees.ipynb` : Carnet principal contenant la connexion à la base de données et les requêtes SQL d'exploration.
* `tableau_de_bord.ipynb` : Analyse graphique et statistiques (Pandas/Matplotlib) basées sur les données extraites.
* `rapport.ipynb` : Synthèse et interprétation des résultats de l'analyse.
* `affichage_pays_compagnies.py` : Script Python générant une table de visualisation stylisée des studios par zone géographique.

## 🛠️ Compétences techniques
* **SQL :** Maîtrise des jointures, agrégations (`GROUP BY`) et filtrages complexes.
* **Data Python :** Utilisation de `Pandas` pour la manipulation de DataFrames et `Matplotlib` pour la dataviz.
* **Anonymisation :** Nettoyage des métadonnées et sorties Jupyter pour la mise en ligne (RGPD/Sécurité).
